### Backend RNG (Python)
```python
import secrets

def generate_craps_result(bet_data: Dict) -> Dict:
    """Cryptographically secure dice roll"""
    dice1 = secrets.randbelow(6) + 1
    dice2 = secrets.randbelow(6) + 1
    total = dice1 + dice2
    
    return {
        "dice": [dice1, dice2],
        "total": total,
        "is_natural": total in [7, 11],
        "is_craps": total in [2, 3, 12]
    }
```
