## 📋 Brand Usage Guidelines

### DO's ✅
- Maintain aspect ratio (don't stretch)
- Use on clean backgrounds
- Maintain clear space (1/4 logo height minimum)
- Use high-resolution version for print
- Keep gradient intact when possible

### DON'Ts ❌
- Don't stretch or distort
- Don't add effects or filters
- Don't change colors (except approved versions)
- Don't place on busy backgrounds
- Don't use low-resolution versions for print
